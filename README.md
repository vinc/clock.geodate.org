# Geodate for WebAssembly

This project builds [geodate](https://github.com/vinc/geodate) written in
rust to wasm and displays it on a webpage.


## Usage

Run:

    $ yarn install
    $ yarn run build
    $ yarn run start

Then visit: <http://localhost:8080>
